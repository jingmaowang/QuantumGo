== 描述 ==

量子围棋游戏集成插件让您可以在 WordPress 网站上轻松集成围棋游戏。用户可以通过点击按钮直接跳转到您的游戏，无需离开网站。

**主要功能：**

* 自动修改 "CogniGo™ (Quantum Go)" 的 "Learn more" 按钮链接
* 添加可自定义的浮动游戏按钮
* 支持自定义按钮文字和位置
* 响应式设计，完美适配移动端
* 点击统计和用户行为分析
* 游戏状态监控
* 完全免费，无需额外费用

**技术特点：**

* 使用 Vercel + Supabase 部署
* 完全免费托管
* 自动扩展和备份
* 全球 CDN 加速
* SSL 证书自动配置

== 安装 ==

1. 下载插件文件
2. 在 WordPress 后台进入"插件" → "安装插件"
3. 点击"上传插件"，选择下载的 ZIP 文件
4. 点击"现在安装"
5. 安装完成后点击"启用插件"

== 配置 ==

1. 在 WordPress 后台进入"设置" → "围棋游戏"
2. 输入您的游戏部署地址（Vercel URL）
3. 自定义按钮文字和位置
4. 选择是否启用浮动按钮
5. 点击"保存更改"

== 使用 ==

**基本使用：**

1. 部署您的围棋游戏到 Vercel
2. 在插件设置中输入游戏 URL
3. 保存设置，插件会自动工作

**高级功能：**

* 浮动按钮：在网站右下角显示游戏按钮
* 自定义位置：选择按钮显示位置
* 响应式设计：自动适配不同设备
* 点击统计：监控用户点击行为
