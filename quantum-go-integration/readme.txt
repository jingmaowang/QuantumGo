在网站首页添加围棋游戏按钮，并自动修改COGNIGO卡片的链接。

== 描述 ==

这个插件会在您的网站首页添加一个固定的围棋游戏按钮，用户点击后可以跳转到您的围棋游戏页面。同时，插件会自动识别并修改COGNIGO™ (QUANTUM GO)卡片的"LEARN MORE"按钮，使其直接跳转到围棋游戏。

功能特点：
* 在首页显示固定的围棋游戏按钮
* 可自定义游戏URL地址
* 可自定义按钮文字和位置
* 自动修改COGNIGO卡片的按钮链接
* 响应式设计，支持移动端
* 美观的渐变背景和悬停效果
* 简单的管理界面

== 安装 ==

1. 将插件文件上传到 `/wp-content/plugins/quantum-go-integration/` 目录
2. 通过WordPress管理后台的"插件"菜单激活插件
3. 在"设置" -> "围棋游戏"中配置游戏URL

== 常见问题 ==

= 如何更改游戏URL？ =

在WordPress管理后台，进入"设置" -> "围棋游戏"，在"游戏URL"字段中输入您的围棋游戏地址。

= 按钮显示在哪些页面？ =

按钮只在网站首页显示，不会影响其他页面。

= 如何自定义按钮样式？ =

您可以在主题的CSS文件中添加自定义样式来覆盖默认样式。

= 如何部署围棋游戏？ =

推荐使用以下云服务平台：
* Vercel：适合前端部署
* Railway：适合全栈应用
* Render：适合后端API
* Netlify：适合静态网站
